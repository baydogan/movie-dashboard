# Movie Dashobard

### About the Project

A movie dashobard app created with Vue 3 Composition API and TMDB API.

### Getting Started

To use API create a file at the root of the project called .env with the following value with the key provided by the TMDB API.

```sh

VITE_MOVIE_APP_KEY=

```

### Installation

```sh
cd movie-dashboard
npm i
npm run dev
```
